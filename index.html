How It Works

Follow these simple steps to start transcribing and summarizing your audio files

Install Required Packages

First, you'll need to install Whisper, OpenAI's Python client, and FFmpeg for audio processing.

Python !pip install git+https://github.com/openai/whisper.git !pip install openai !apt update && apt install -y ffmpeg
1

Upload Your Audio Files

Use Google Colab's file uploader to add your MP3 or WAV files to the session.

Python from google.colab import files uploaded = files.upload()
2

Transcribe with Whisper

Use OpenAI's Whisper model to convert the audio to text with high accuracy.

Python import whisper model = whisper.load_model("base", device="cpu") transcripts = {} for filename in uploaded.keys(): result = model.transcribe(filename) transcripts[filename] = result["text"]
3

Summarize with GPT-4

Send the transcripts to OpenAI's GPT-4 API to generate concise, meaningful summaries.

Python import openai openai.api_key = "sk-..." # Replace with your API key summaries = {} for filename, transcript in transcripts.items(): prompt = f"Please summarize the following transcription:\n\n{transcript}" response = openai.ChatCompletion.create( model="gpt-4", messages=[ {"role": "system", "content": "You're a helpful assistant that summarizes transcripts."}, {"role": "user", "content": prompt} ] ) summary = response.choices[0].message.content.strip() summaries[filename] = summary
4

Save and Download Results

Generate text files with your summaries and download them to your local machine.

Python for filename, summary in summaries.items(): outname = filename.rsplit(".", 1)[0] + "_summary.txt" with open(outname, "w") as f: f.write(summary) files.download(outname)
5
